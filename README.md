# 🌐 Mastek Innovation – Frontend

This is the frontend of the **Mastek Innovation** website built using **React + Vite**. It provides a modern, responsive UI for showcasing services, careers, and contact functionality.

## 🚀 Live Demo

🔗 [live link](https://onesoftware-company-1.onrender.com) 

## 🔗 Backend Repository

👉 [backend repo](https://github.com/saadparekh/2software_company)

## 🛠️ Tech Stack

* React (Vite)
* React Router DOM
* Tailwind CSS / CSS
* Axios (for API calls)

## 📂 Features

* Multi-page routing (Home, Careers, Services)
* Fully responsive design
* Contact form integration
* Career application form with resume upload
* Clean UI with reusable components (Navbar, Footer, Topbar)

## 📁 Project Structure

```
src/
 ├── components/
 ├── pages/
 │    ├── services/
 ├── App.jsx
 └── main.jsx
```

## ⚙️ Installation & Setup

```bash
# Clone the repo
git clone https://github.com/your-username/frontend-repo.git

# Navigate into project
cd frontend-repo

# Install dependencies
npm install

# Run development server
npm run dev
```

## 🔌 API Integration

This frontend connects with the backend APIs:

* POST `/contact`
* POST `/career`
